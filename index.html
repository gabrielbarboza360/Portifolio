Conhecimento

HTML é uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir.

CSS é uma linguagem de folha de estilo composta por "camadas", criando com o propósito de estilizar as paginas HTML

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

Pratica básica em estilização de imagem .
